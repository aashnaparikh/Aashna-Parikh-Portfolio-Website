# Aashna Parikh - Portfolio Website

A modern, interactive portfolio website showcasing my journey as a Computer Science student at the University of Victoria. This site highlights my technical projects, professional experience, and passion for technology and empowering women in tech.

## Features

- **Dynamic Hero Section** - Animated typing effect showcasing my roles and interests
- **Interactive Timeline** - Comprehensive experience section displaying 10+ professional experiences
- **Particle Background** - Engaging visual effects with animated particles
- **Geometric Animations** - Astrology-inspired geometric patterns
- **Smooth Navigation** - Seamless scrolling between sections
- **Responsive Design** - Fully optimized for all device sizes
- **Custom Theme** - Deep purple aesthetic (#1a0033, #bb86fc) with glowing accent effects

## Tech Stack

- **Framework**: React.js (Create React App)
- **Styling**: CSS3 with custom animations
- **Icons**: Lucide React
- **Typography**: NTR Font Family
- **Version Control**: Git & GitHub
- **Deployment**: GitHub Pages / Vercel / Netlify (TBD)

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone https://github.com/aashnaparikh/Aashna-Parikh-Portfolio-Website.git
cd Aashna-Parikh-Portfolio-Website
```

2. Install dependencies
```bash
npm install
```

3. Start the development server
```bash
npm start
```

4. Open [http://localhost:3000](http://localhost:3000) to view it in your browser
